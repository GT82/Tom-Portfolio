Responsive Web Design Cheat Sheet

Published in Tutorials, design tips, User Experience

How to set up breakpoints in this design? You probably ask yourselves this question a lot.

Responsive design breakpoints is no rocket science and you don’t need to search the web for the proper breakpoint dimensions every time you design for responsive. As a matter of fact, the ultimate design aid in this matter would be a simple graph with all the breakpoint dimensions visualised.

And that’s what we give you.

Using the responsive web design cheat sheet is very simple.Find the device you’re designing for on the chart and read what breakpoint dimensions it should have – and there you have it.

The poster is print-ready so you can even put it on your wall and never lose track of responsive dimensions again (or hang it in your office and make your teammates stop haunting you with questions about breakpoints ;) ).

You can download the responsive web design cheat sheet here (102 kB).

P.S. In UXPin, the UX Design App, you can set up breakpoints using our quick ready-made breakpoints for most popular devices or set up your own using custom breakpoints feature. Sign up for a free trial and try it!
